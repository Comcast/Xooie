Addons

Addons allow you to add additional functionality to a widget without having to create a whole new widget type. This works well if you need to tweak functionality or add new features.

Instantiating Addons

Addons can be instantiated using the data-addons property. Multiple addons can be loaded at the same time, separated by a space:


        <div data-widget-type="tab" data-addons="tab-animation">
            <ul>
                <li><a>Tab 1</a></li>
                <li><a>Tab 2</a></li>
                <li><a>Tab 3</a></li>
            </ul>

            <div>Content for Tab 1</div>

            <div>Content for Tab 2</div>

            <div>Content for Tab 3</div>
        </div>

Creating Addons

Addons are created using the extend method. Instead of extending widgets, addons extend from other addon modules:


        define('my_addon', ['jquery', 'addons/base'], function($, Addon){
            var MyAddon = Addon.extend(function(){
                // constructor functionality
            });

            return MyAddon;
        });
