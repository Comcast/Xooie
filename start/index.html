Getting Started With Xooie

Getting started with Xooie is as easy as 1, 2, 3...

1. Set a widget type

Pick a widget type and set the data-widget-type attribute on the root element of your widget:


        <div data-widget-type="tab">
            <ul>
                <li><a>Tab 1</a></li>
                <li><a>Tab 2</a></li>
                <li><a>Tab 3</a></li>
            </ul>

            <div>Content for Tab 1</div>

            <div>Content for Tab 2</div>

            <div>Content for Tab 3</div>
        </div>

2. Define your roles

Add the appropriate data-x-role attributes to the key elements in your widget:


        <div data-widget-type="tab">
            <ul data-x-role="tablist">
                <li><a data-x-role="tab">Tab 1</a></li>
                <li><a data-x-role="tab">Tab 2</a></li>
                <li><a data-x-role="tab">Tab 3</a></li>
            </ul>

            <div data-x-role="tabpanel">Content for Tab 1</div>

            <div data-x-role="tabpanel">Content for Tab 2</div>

            <div data-x-role="tabpanel">Content for Tab 3</div>
        </div>

3. Test your page

Xooie widgets automatically instantiate when you load the page, so there's no JavaScript code to write. It's that easy!
